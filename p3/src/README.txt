== Build ==
build.xml is provided in the root folder, use ant to build the project.

== Blog using rule ==
1 Number signs are used at the beginning and the end of text to specify a tag (e.g #tag#).
2 As initialisation a publish topic is subscribed.
3 Both user names and tags are regarded as topics. Follow label will show up for convenient subscription by clicking the user name, user avatar or highlighted tag.
